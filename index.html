Dip Wallet - Electronic Wallet

DIP Wallet is an application that is aimed to revolutionize Indonesia's money and reward saving system. In time when everything has been digitalized, we would like simplify people's lifestyle; including being able to leave your wallet at home and only bringing your phone with you. Users will be able to store their credit cards and reward cards information digitally and retrieve it when needed. Working together with several retailers, user will be able to get reward points when they use DIP Wallet to purchase goods. Points can be redeemed for discount or free samples.


This application was on going throught out my summer internship. The first prototype was to be tested on Android phones so we used Android Studio for the mobile layout structure and Java as the main programming language. Photoshop was used to make the application graphics and icons while user informations were stored in SQL database. The project is still active and is currently on testing phase before released to the public.


Technology: Photoshop, Java, Android Studio, SQL

Hommi - House-searching website

Hommi is a website that is used for housing and housemates searching. The idea was first directed towards the needs of UCSD students of off-campus housing. Besides the usual filter features that user can use to search for their ideal housing environment, the website also offer housemate 'filter' that can help match user with their ideal housemates. We hope to help users to not only find the best housing, but also find a friend to live with.


This website was developed over the summer course session together with 5 other teammates. We started looking for idea and feel that this website might be a solution for many students' problem. We developed the persona and storyboard to see what problems we are trying to tackle with the creation of the website. We did the first draft of the website design with paper and pencil then did the first prototype so we can do user testing. After many input, comments, recommendations, and discussion, we did more prototyping and finally came out with our final design.


Technology: Balsamiq, Axure

TBA - Textbook buying and selling app

TBA is a mobile application that can be used to buy and sell used textbook. Students across the country will be able to post a listing to sell their textbook. They can also look for books that they want and contact the owner to discuss the price and way of delivery. Buyers get to choose which listing they would buy books from based on the book condition, price, and distance of the owner. This application was made by students and for students in mind.


TBA was developed by a group of 10 students over the course of 10 weeks using Agile methodology. The application was made using Android Studio as the primary IDE. We linked the application with an active ISBN Library so that we can cross-checked the validity of the information entered by seller. Parse is used to handle user account info. During the development, the team held a weekly presentation to monitor the project progress and ask for inputs and comments.

Technology: Parse, Android Studio, Java, ISBN Library
